# Демо-версия разработанного мной календаря.

## Данный проект представлят собой календарь, с возможностью добавлять события, реализованный с помощью React, Redux, Bootstrap и других технологий.

## Функционал

- Добавление событий
- Изменение режима отображения
- Анимация сворачивания и разворачивания
- Покрытие unit-тестами

## Технологии

- **React**
- **Redux**
- **Bootstrap**
- **Framer motion**
- **Prettier & ESLint**
- **Vitest + React Testing Library**
